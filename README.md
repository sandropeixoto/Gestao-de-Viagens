# Gestao de Viagens

## Project Description
Gestao de Viagens is a web application that facilitates the management of travel plans and itineraries. It allows users to create, view, and edit trips seamlessly, ensuring an organized travel experience.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT
- Deployment: Heroku

## Features
- User authentication and authorization
- Create, read, update, and delete trips
- Interactive itinerary management
- User-friendly interface
- Responsive design for mobile and desktop

## Quick Start Guide
1. Clone the repository:
   ```bash
   git clone https://github.com/sandropeixoto/Gestao-de-Viagens.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Gestao-de-Viagens
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000` to view the application.
